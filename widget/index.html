Free crypto sentiment widget

Drop live crypto social-sentiment data on your site in one line. No API key, no signup, free forever. Real-time data by socialtickers.

Install

<script src="https://paperclip-calc.zivtools.workers.dev/widget.js" data-coins="BTC,ETH,XRP"></script>

Change data-coins to any tickers. The widget renders a compact sentiment box wherever you place the script.
